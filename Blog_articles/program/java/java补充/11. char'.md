1.char表示字符，定义时用单引号，只能存储一个字符，如char c=’x’;
而String表示字符串，定义时用双引号，可以存储一个或多个字符，如String name=”tom”;

2.char是基本数据类型，而String 是一个类，具有面向对象的特征，可以调用方法，如name.length()获取字符串的长度。
